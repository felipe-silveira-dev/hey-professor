## About Hey Professor
An application to help teachers to manage their classes and students.

## Installation

```bash
cp .env.example .env
```

```bash
composer install
```

```bash
php artisan key:generate
```

```bash
php artisan migrate --seed
```
```bash
npm install
```

## Usage

```bash
php artisan serve
```

```bash
npm run dev
```
